# VISU_DONNEES_INFO5
TPs de Visualisation de Données en INFO5 à Polytech Grenoble

Réalisés par Akram Roubia, Arthur Millet et Pierre Bernard

## Lancement des TPs :
 * Démarrer le serveur dans l'archive : python3 simple-cors-http-server.py
 * Ouvrir un navigateur avec l'URL : http://localhost:8000
 * Choisir le TP puis naviguer sur les différentes pages
